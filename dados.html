Nossas fontes

Transparência

Tudo aqui foi feito com software de código aberto e está disponível para livre uso 1.

Os dados e os códigos-fonte deste site estão em repositório público (https://github.com/cantareira). Quem conhece as ferramentas computacionais que usamos pode verificar os resultados ou realizar outras análises.

Para os que não dominam estas ferramentas computacionais, as seções abaixo disponibilizam os dados:

Volume e pluviosidade nos reservatórios de SP de 01/01/2003 a 21/01/2020

  • Fonte: Boletins diários da SABESP de situação dos manaciais, recuperados automaticamente com a ferramenta mananciais.tk
  • Arquivo de dados
  • Metadados
  • Notas:
    • Podem conter erros de leitura devido ao processo automático de recuperação dos dados.
    • Nestes dados originais os volumes relativos não estão corrigidos para o acréscimo devido à incorporação do volume morto. Isso foi corrigido nos dados usados nas análises.

Vazões de entrada e saída no sistema Cantareira de 01/01/2004 a 21/01/2020

  • Fonte: Boletins diários da SABESP de vazão do sistema, recuperação semi-automatizada.
  • Arquivo de dados
  • Metadados
  • Podem conter erros de leitura devido ao processo semi-automático de recuperação dos dados.

Dados processados usados em nossas análises (01/01/2004 a 21/01/2020)

  • Fonte: dados de volume, pluviosidade e vazões acima.
  • Arquivo de dados
  • Metadados
  • Nota: valores discrepantes foram retirados, valores faltantes foram interpolados linearmente e quantidades derivadas necessárias para as projeções foram calculadas.

Projeções de volume de água armazenada até 20/02/2020

  • Fonte: projeções para os cenários de chuva na média histórica, 25% a menos e 25% a mais que esta média
  • Arquivo de dados
  • Metadados
